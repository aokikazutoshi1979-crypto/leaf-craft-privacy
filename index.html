プライバシーポリシー

アプリ「Leaf Craft 温活コンテンツ投稿」における個人情報の取り扱いについて

運営者：合同会社Leaf Craft（楽天市場店「LaforetMarket」運営元）
制定日：2026年7月4日

本アプリは、運営者が自ら管理する Pinterest ビジネスアカウントに、自社の温活・冷え対策に関するお役立ちコンテンツ（画像・説明文）を投稿・管理するためのツールです。他の Pinterest 利用者やウェブサイト訪問者の個人情報を収集・保存・利用することはありません。

1. 本アプリの目的

本アプリは、Pinterest API を利用して、運営者自身の Pinterest アカウントに対してピンの作成・投稿・整理を行い、また当該アカウントの投稿実績（表示回数・保存・クリック等の統計）を取得します。これらはすべて運営者自身のアカウントに関する操作に限られます。

2. 取得・取り扱う情報

  • 運営者自身の Pinterest アカウントへアクセスするための認証情報（アクセストークン）。
  • 運営者自身のボード・ピン、およびそれらの投稿実績（統計データ）。

本アプリは、上記以外の第三者（他の Pinterest 利用者・一般の閲覧者）の氏名・連絡先・閲覧履歴などの個人情報を、収集・保存・処理することは一切ありません。

3. 情報の利用目的

取得した情報は、次の目的にのみ利用します。

  • 運営者自身の Pinterest アカウントへのコンテンツ投稿および整理。
  • 投稿したコンテンツの効果測定と、今後の発信内容の改善。

4. 第三者への提供

本アプリが取り扱う情報を、第三者へ販売・提供・共有することはありません。広告目的での第三者提供も行いません。

5. Pinterest API の利用

本アプリは Pinterest の開発者向け規約に従って Pinterest API を利用します。取得したアクセストークンは、運営者が管理する安全な保管領域（暗号化されたシークレット管理サービス）に保存し、本アプリの動作以外の目的には利用しません。

6. データの保管とセキュリティ

認証情報は暗号化して保管し、アクセス権限を運営者に限定します。不要になった認証情報は速やかに失効・削除します。

7. Cookie・トラッキング

本アプリは、閲覧者を追跡する Cookie やトラッキング技術を使用しません。

8. ポリシーの改定

本ポリシーは、法令の変更や運用の見直しに応じて改定することがあります。重要な変更がある場合は、本ページ上で更新します。

9. お問い合わせ

本ポリシーおよび個人情報の取り扱いに関するお問い合わせは、info@leaf-craft.com または楽天市場の当店舗（LaforetMarket）ページ内のお問い合わせ窓口よりご連絡ください。
